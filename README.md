# MINI ML-DL PROJECTS

A curated collection of **Machine Learning (ML)** and **Deep Learning (DL)** mini-projects built for academic learning, portfolio building, and hands-on experimentation.  
This repository demonstrates reproducible pipelines, modular code, and practical implementations of ML/DL algorithms.

---

## 📂 Repository Structure

- **DL-(DEEP_LEARNING)** → CNN, RNN, GAN projects  
- **ML-(MACHINE_LEARNING)** → ML algorithms (regression, classification, clustering)  
- **.gitattributes** → Git attributes for handling text/binary files  

---

## 🚀 Features

- **CNN Projects** → Image classification pipelines  
- **RNN Projects** → Sequence modeling (text/time-series)  
- **GAN Projects** → Generative models for synthetic data  
- **ML Algorithms** → Regression, decision trees, clustering, etc.  
- **Visualization** → HTML-based project visualization

# MINI_ML-DL_PROJECTS 

##  Overview
A collection of **mini ML/DL projects** for hands‑on practice and portfolio building.  
Includes classical ML algorithms and modern DL models like **CNNs, RNNs, and GANs**.  

---

## Features
- Text Summarization (Hugging Face Transformers, T5)  
- Stock Price Prediction (KNN)  
- GPA Prediction (Random Forest)  
- Breast Cancer Classification (ML models)  
- Wine Quality Classification (Regression dataset)  
- General Knowledge Q&A (RNN)  
- Interactive Web Apps (Gradio & FastAPI)  
- Modular, Reusable, Docker‑Ready  

---

## Tech Stack
- **Languages:** Python, HTML  
- **Libraries:** FastAPI, Gradio, Transformers, PyTorch, Scikit‑learn, Pandas, NumPy  
- **Tools:** Jupyter Notebooks  

---

## Installation
```bash
git clone https://github.com/promedchandradas4/MINI_ML-DL_PROJECTS.git
cd MINI_ML-DL_PROJECTS
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r DL-(DEEP_LEARNING)/NLP/Text_summarizer/requirements.txt
```

Run a project (example: Text Summarizer):
```bash
cd DL-(DEEP_LEARNING)/NLP/Text_summarizer
python app.py
```
Open `http://localhost:7860` in your browser.
